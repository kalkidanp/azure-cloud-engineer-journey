## Overview 🎯
Today I reviewed everything I learned this week and focused on understanding how all components connect in a complete system.

---

## My Understanding 🧠

- User traffic enters through Application Gateway  
- Routed to backend services (API / Images)  
- Backend protected using NSG  
- Admin access secured via Azure Bastion  
- Data stored in Azure Storage  
- Access secured via Private Endpoint  
- DNS ensures proper internal resolution  

---

## What I Realized 💡
- Cloud architecture is about connections, not individual services
- Security must exist at multiple layers
- Traffic flow design is critical
- Even small misconfigurations can break the system
